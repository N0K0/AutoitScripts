What is Pytomatic

Pytomatic is a system i'm currently developing to automate tasks with Python. Its main focus is being a general purpose package for mainly games and other mouse-driven games. Like for example BF4 commander mode (Which is how this all started)

Features

  • Basic Window management
  • Basic Mouse control
  • Basic Image analysis
